Listas HTML

Listas ordenadas

Chamamos de ordered lists todas as listas que podemos ordena-las.
Para criar uma ordered lists, vamos usar a tag <ol> para delimitar e <li> para identificar cada item da lista.

  1. Item 1
  2. Item 2
  3. Item 3
Podemos alterar o marcador utilizado o atríbuto type.
Utilizando um dos seguintes marcadores: 1, A, a, I, i.

Listas Não Ordenadas

unordered lists são listas apenas com marcadores.
Utilizamos <ul> para delimitar e <li> para identificar

  • Item 1
  • Item 2
  • Item 3
Podemos alterar o marcador utilizado o atríbuto type.
Utilizando um dos seguintes marcadores: disc, circle ou square.

Lista de Definições

Toda lista de definição está dentro de uma tag <dl> (definition list). Cada termo é um <dt> (definition term) e cada descrição é um <dd> (definition description).
